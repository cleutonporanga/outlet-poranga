# **App Name**: ModaEstoque Pro

## Core Features:

- Gerenciamento de Produtos: Permite adicionar, editar, visualizar e excluir produtos, incluindo variações de tamanho e cor. Usa uma tela modal moderna para cadastro de novos itens.
- Controle Automático de Estoque: Atualiza automaticamente a quantidade do estoque para cada produto após entradas ou saídas e dispara alertas visuais para itens com estoque baixo (quantidade < 5).
- Histórico de Movimentações: Exibe um registro cronológico de todas as entradas e saídas de estoque, indicando o tipo de movimentação (entrada/saída), data e quantidade.
- Painel de Controle (Dashboard): Apresenta um resumo executivo com cards informativos sobre o total de produtos, itens em estoque baixo e o valor total do inventário, complementado por um gráfico simples.
- Relatórios Essenciais: Gera relatórios simplificados, como um resumo geral do inventário e uma lista dedicada aos produtos com baixo estoque para ações rápidas.
- Busca e Filtro de Produtos: Funcionalidade de busca integrada e opções de filtro para localizar e organizar produtos rapidamente na lista de estoque.
- Assistente de Descrição de Produto com IA: Utiliza uma ferramenta de IA para sugerir descrições atraentes e detalhadas para produtos novos ou existentes, baseando-se no nome, categoria, tamanho e cor fornecidos.

## Style Guidelines:

- Primary color: Muted deep green (#394C3B) for professional main text and key UI elements, offering strong contrast against light backgrounds.
- Background color: A very subtle, almost off-white greenish tint (#F4F7F4) for a clean and light canvas, enhancing readability and a modern feel.
- Accent color: Vibrant green (#2E7D32) derived from the user's theme for interactive elements, highlights, tab indicators, and calls to action, providing visual focus.
- Primary font: 'Inter', a grotesque-style sans-serif, for both headlines and body text, ensuring a modern, neutral, clean, and highly legible interface.
- Utilize a consistent set of modern, minimalist outline icons for navigation, actions, and informational graphics, matching the clean and professional aesthetic.
- Implement top horizontal tab navigation (Top Tab Navigation) with a distinct green line indicator below the active tab. All content within tabs will utilize modern cards with soft shadows and rounded borders for readability and visual appeal. Include a floating action button (+) for product additions and ensure a fluid, responsive design across various mobile screen sizes.
- Incorporate smooth, subtle transition animations when switching between tabs via taps or horizontal swipes, along with brief visual feedback (e.g., slight elevation or ripple) when saving data or interacting with elements to enhance user experience.
